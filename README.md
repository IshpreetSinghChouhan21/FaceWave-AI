# AI Gesture + Face & Emotion Recognition

Real-time AI project using MediaPipe + OpenCV that:
- Detects hand gestures
- Counts fingers
- Detects emotions (Happy / Neutral / Surprised)
- Recognizes faces and remembers them between runs

## How to Run
1️⃣ Install dependencies  
pip install -r requirements.txt

2️⃣ Run  
python3 gesture.py

## Face Registration
Press R while your face is visible, enter your name in terminal.

## Privacy
No face images are uploaded. Face data is stored only locally.
